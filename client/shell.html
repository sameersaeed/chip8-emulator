CHIP-8 Emulator

A simple CHIP-8 emulator written in C++ and compiled to WebAssembly.
Use the dropdown to select a ROM and press the play button to start the emulator.

You can also use the controls mentioned below to interact with the available ROMs
For more information on this project, check out the GitHub!
